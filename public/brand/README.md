# Join Care Now — brand assets

Drop the logo PNG files into this folder (provided separately). Filenames and intended use:

- `jcn-logo-full-transparent.png` — primary logo, transparent bg. Use on light backgrounds (site header).
- `jcn-logo-full-white.png` — logo on a solid white tile.
- `jcn-logo-reversed.png` — for DARK backgrounds (footer, dark sections): coloured heart-house mark + white wordmark.
- `jcn-logo-white-mono.png` — single-colour white version, for photos/overlays.
- `jcn-icon-512.png`, `jcn-icon-192.png`, `jcn-icon-180.png`, `jcn-icon-32.png` — the heart-house mark only, for favicon and app/touch icons.
- `jcn-mark-transparent.png` / `jcn-mark-white.png` — the mark on its own.

Notes:
- These are raster (PNG) assets cut from a 1254px master — crisp for web, favicons and social. For large print/signage, use the original vector (AI/SVG/EPS) if available.
- Brand colours (from the site): primary teal #0d9488, secondary cyan #0e7490, accent indigo #3730a3, page bg #f9fafb, text #111827.
